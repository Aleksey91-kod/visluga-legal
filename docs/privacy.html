Политика конфиденциальности мобильного приложения «Выслуга+»

Дата обновления: 2025 г.

1. Общие положения

Настоящая Политика регулирует обработку информации в мобильном приложении «Выслуга+». Приложение работает автономно и не передает данные третьим лицам.

2. Обрабатываемые данные

Приложение не собирает персональные данные пользователей. Сохраняются только локальные данные:

  • график смен;
  • периоды отпусков;
  • данные расчёта выслуги;
  • настройки приложения.

3. Передача данных

Приложение не передает данные в интернет и не использует сторонние сервисы.

4. Реклама и аналитика

В текущей версии приложение не содержит рекламы и не использует аналитические SDK.

5. Разрешения

Приложение не запрашивает доступ к камере, микрофону, контактам, геолокации и файловой системе.

6. Удаление данных

Все данные хранятся локально. Для удаления достаточно удалить приложение с устройства.

7. Контакты

По вопросам работы приложения можно обратиться через RuStore.
